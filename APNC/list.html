Type of list

  • Unordered List
  • Ordered List

This is example of unordered list.

    Name of fruits:
  • Appke
  • Banana
  • Mango
  • Orange

This is example of ordered list.

    Name of animals:
  1. Ape
  2. Buffalo
  3. Cat
  4. Dog

Attribute of list

We can use type="types" attribute in list to change the types of list in ordered list. This can only be use on ordered list.
    Name of Gods:
  • Bramha
  • Vishnu
  • Mahesh
